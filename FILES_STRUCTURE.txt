════════════════════════════════════════════════════════════════════════════
                          FILE DEPENDENCIES & FLOW
════════════════════════════════════════════════════════════════════════════

Configuration Flow:
  umbrella_config.yaml
          ↓
  master_umbrella_setup.py (ConfigLoader)
          ↓
  All other modules read via config_get()

Server Flow:
  rat_server_fixed.py
          ├─→ master_umbrella_setup.py (config)
          ├─→ agent_registry.py (database)
          ├─→ communication_managers.py (heartbeat)
          ├─→ rest_api_server.py (Flask API)
          └─→ command_executor.py (command execution)

Bot Flow:
  whatsapp-c2/bot.js
          ├─→ utils/apiBridgeClient.js (REST API calls)
          ├─→ utils/apiCommandHandlers.js (process commands)
          ├─→ utils/configLoader.js (load config)
          └─→ commands/*.js (execute specific commands)

Agent Builder Flow:
  rat_executable_builder.py
          ├─→ rat_ultimate.py (agent code)
          ├─→ master_umbrella_setup.py (config)
          └─→ Outputs to build/
